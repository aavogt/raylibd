# Raylibd

    The raylibd program
    
    raylibd [COMMAND] ... [OPTIONS]
    
    Common flags:
      -? --help               Display help message
      -V --version            Print version information
    
    raylibd [raylibd] [OPTIONS] [FILE]
      turn [FILE]
    
         --outputmain=ITEM  
         --cflags=ITEM        c preprocessor flags
         --cflags-extra=ITEM
      -e --echo             
         --once             
    
    raylibd init [OPTIONS] ITEM
    
      -f --force              overwrite files
